## Core Mapping Principles

### The Three Dimensions of Musical Encoding

1. **Harmonic** (Key/Chords) = **Emotional State**
2. **Rhythmic** (Tempo/Meter) = **Activity Type**
3. **Melodic** (Notes/Intervals) = **Specific Actions**

### Key-Emotion Mapping

| Key | Emotional State | When to Use |
|-----|----------------|-------------|
| **G major** | Stable, grounded, triumphant | Success, confidence, feature complete |
| **C major** | Clear, simple, accessible | Straightforward solutions, clarity |
| **D major** | Bright, structural | Architectural work, Nyro's analysis |
| **E minor** | Searching, uncertain, investigative | Bug discovery, confusion |
| **A minor** | Melancholic, reflective | Debugging, introspection |
| **D minor** | Struggle, pain, challenge | Performance issues, hard problems |

### Tempo-Activity Mapping

| Tempo | BPM Range | Activity Type | Dynamics |
|-------|-----------|--------------|----------|
| **Largo** | 40-60 | Session completion, peaceful rest | pp-ppp |
| **Adagio** | 60-76 | Bug discovery, confusion, documentation | pp-p |
| **Andante** | 76-108 | Planning, research, investigation | p-mp |
| **Moderato** | 108-120 | Active coding, steady work | mf |
| **Allegro** | 120-156 | Testing, building, excitement | f |
| **Vivace** | 156-176 | Tests passing, deployment | ff |
| **Presto** | 176-200 | Jerry's joy, extreme celebration | fff |

### Interval-Meaning Mapping

| Interval | Meaning | Usage |
|----------|---------|-------|
| **Perfect 5th** (G→D) | Structural foundation, stability | Nyro's lattice, architecture |
| **Perfect 4th** (C→F) | Solid progression, building | Implementation steps |
| **Major 3rd** (C→E) | Positive, uplifting | Success moments |
| **Minor 3rd** (E→G) | Reflective, uncertain | Investigation |
| **Minor 2nd** (C→C#) | Tension, dissonance | Bugs, errors |
| **Tritone** (C→F#) | Maximum tension, crisis | Critical bugs, major problems |
| **Octave** (C→c) | Unity, completion | Full circle, resolution |
| **Ascending scale** | Building, growing | Feature implementation |
| **Descending scale** | Calming, resolving | Completion, winding down |
| **Chromatic** | Searching, uncertain | Debugging, exploration |
