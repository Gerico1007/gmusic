## What Are Leitmotifs?

**Leitmotif** (German: "leading motif") - A short, recurring musical phrase associated with a particular person, place, concept, or emotion. When that element appears in the story, its musical theme plays.

**In G.Music Assembly:** Each Assembly member, code event, and emotional state has a unique musical signature that recurs across sessions, creating a **unified musical language** for expressing programming journeys.

## Usage Guidelines

### How to Use These Leitmotifs

1. **Identify the Moment**: What's happening in the code journey?
2. **Choose the Leitmotif**: Match the character/event/emotion
3. **Adapt to Context**: Transpose key, adjust tempo/dynamics
4. **Combine Leitmotifs**: Layer multiple motifs for complex moments
5. **Develop the Theme**: Augment, diminish, invert, transform

### Leitmotif Development Techniques

**Augmentation** (lengthen durations):
```abc
% Original Nyro motif
"G"G2D2 | D2G2 |
% Augmented (slower, emphasized)
"G"G4D4 | D4G4 |
```

**Diminution** (shorten durations):
```abc
% Original
"G"G2B2 d2g2 |
% Diminished (faster, urgent)
"G"GBGB dgdg |
```

**Inversion** (flip melody):
```abc
% Original (ascending)
"G"G2B2 d2g2 |
% Inverted (descending)
"G"g2d2 B2G2 |
```

**Transposition** (different key):
```abc
% Nyro in G major
"G"G2D D2G |
% Nyro in D major (different context)
"D"D2A A2D |
```

### Combining Multiple Leitmotifs
